Testing a Website


During one of my courses, I created a test suite to test a local page of login. In it, I assert values of of the webpage bieng via CSS selector like checkboxes, blank spaces, login, the privacy policy, simulate drag-and-drop of files, make an HTTP request and even use a custom command. All the tests were written in JavaScript.




    Website:
  1. Checks The Application Title
  2. Fills In The Required Fields And Submits The Form
  3. Displays Error Message When Submitting The Form With An Email With Invalid Formatting
  4. The Phone Field Remains Empty When Filled With A Non-Numeric Value
  5. Displays Error Message When The Phone Number Becomes Mandatory But Is Not Filled In Before Sending The Form
  6. Fills In And Clears The First Name, Last Name, Email And Phone Fields
  7. Displays Error Message When Submitting The Form Without Filling In The Required Fields
  8. Successfully Submits The Form Using A Custom Command
  9. Selects A Product (Youtube) By Text
  10. Selects A Product (Mentorship) By Its Value
  11. Selects A Product (Blog) By Its Index
  12. Selects The Type Of Service "Feedback".
  13. Ticks Each Type Of Service
  14. Check Both Checkboxes, Then Uncheck The Last One
  15. Selects A File From The Fixtures Folder
  16. Select A File Simulating A Drag-And-Drop
  17. Select A File Using A Fixture That Has Been Given An Alias
  18. Verify That The Privacy Policy Opens In Another Tab Without A Click
  19. Accesses The Privacy Policy Page By Removing The Target And Then Clicking On The Link
  20. Tests The Privacy Policy Page Independently
  21. Displays And Hides Success And Error Messages Using .Invoke
  22. Fills The Text Area Using The Invoke Command
  23. Makes An Http Request
  24. By Finding A Cat
